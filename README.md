# Personal Portfolio

A responsive portfolio website built with React to showcase my projects, skills, education, certifications, and contact information.

## Live Demo

🔗 https://portfolio-8t5q.vercel.app

## Features

- Responsive design
- Mobile navigation menu
- Smooth scrolling
- Project showcase
- Downloadable resume
- Certifications section
- Contact section

## Technologies Used

- React
- CSS3
- JavaScript
- Vite
- Git
- GitHub
- Vercel

## Installation

git clone https://github.com/sonaliacharya13/portfolio.git

cd portfolio

npm install

npm run dev

## Deployment

Deployed on Vercel.
